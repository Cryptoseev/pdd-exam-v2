# Дизайн: Голос, Объяснение, Разбор ошибок

## Цель
Добавить три функции в приложение «Экзамен ПДД» для Нины Леонидовны:
1. Голосовое озвучивание вопросов
2. Кнопка «Объяснить»
3. Экран разбора ошибок после экзамена

---

## 1. 🔊 Голосовое озвучивание

### Поведение
- При открытии каждого вопроса автоматически запускается озвучивание
- Читается: текст вопроса, затем варианты ответов по очереди
- Язык: `ru-RU`
- При переходе на следующий вопрос — текущее чтение останавливается
- Кнопка ⏸/▶ рядом с вопросом для паузы/продолжения

### Настройки
- В разделе «Настройки» переключатель «Автозвук» (по умолчанию включён)
- Добавить поле `autoVoice: boolean` в тип `Settings`

### Технология
- Web Speech API (`window.speechSynthesis`) — офлайн, бесплатно
- Хук `useVoice(text: string, enabled: boolean)` — инкапсулирует логику

### Файлы
- Создать: `hooks/useVoice.ts`
- Изменить: `components/QuestionCard.tsx` — добавить кнопку и вызов хука
- Изменить: `app/settings/page.tsx` — новый переключатель
- Изменить: `types/index.ts` — поле `autoVoice` в `Settings`

## 2. 🧠 Кнопка «Объяснить»

### Поведение
- Кнопка «Объяснить» отображается под вариантами ответов
- Активна только после того как выбран ответ
- При нажатии раскрывается блок с текстом из поля `explanation` вопроса
- Если `explanation` пустое — показывается «Объяснение недоступно»

### UI
- Кнопка: серая, текст «💡 Объяснить»
- Блок объяснения: светло-жёлтый фон, иконка лампочки, текст

### Файлы
- Изменить: `components/QuestionCard.tsx` — добавить кнопку и блок объяснения

## 3. 📊 Экран разбора ошибок

### Поведение
- На экране результатов (`/results`) появляется кнопка «Разбор ошибок»
- Кнопка видна только если есть хотя бы одна ошибка
- Ведёт на новый экран `/review`
- На экране `/review` показываются только вопросы где ошиблась Нина Леонидовна

### Данные
- Берутся из `sessionStorage` ключ `lastAttempt` (уже есть)
- Дополнительно сохраняются детали ответов: `sessionStorage` ключ `lastAttemptAnswers`
  - Формат: `{ questionId: string, selectedAnswer: number, correctAnswer: number }[]`

### По каждой ошибке показывать
- Текст вопроса
- ❌ Выбранный ответ (красный)
- ✅ Правильный ответ (зелёный)
- Объяснение (из поля `explanation`)

### Файлы
- Создать: `app/review/page.tsx`
- Изменить: `app/results/page.tsx` — кнопка «Разбор ошибок»
- Изменить: `app/exam/page.tsx`, `app/tickets/[id]/page.tsx`, `app/random/page.tsx`, `app/mistakes/page.tsx` — сохранять детали ответов в `sessionStorage`
